# 📱 Social Media Platform

A modern social media web application built using HTML, CSS, JavaScript, Node.js, and Express.js.

## 🚀 Features

* User Profile Section
* Create Posts
* Like Posts
* Comment on Posts
* Interactive User Interface
* Responsive Design

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js

## 📂 Project Structure

social-media-platform

├── public

│ ├── index.html

│ ├── style.css

│ └── script.js

├── server.js

├── package.json

└── .gitignore

## ▶️ How to Run

1. Clone the repository
2. Install dependencies

npm install

3. Start the server

node server.js

4. Open

http://localhost:5000
